# A/B Testing & Statistical Analysis

## Project Overview
This project analyzes the performance of two website versions using A/B Testing. The objective is to determine whether the new website design (Group B) performs significantly better than the old website design (Group A).

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- SciPy

## Dataset
The dataset contains:
- User ID
- Group (A/B)
- Conversion Status

## Steps Performed
- Imported the dataset
- Checked dataset information
- Checked missing values
- Generated summary statistics
- Calculated conversion rates
- Visualized conversion rates using a bar chart
- Performed Chi-Square Test
- Interpreted the p-value

## Results
- Group A Conversion Rate: 46.67%
- Group B Conversion Rate: 80.00%
- P-value: 0.1297

## Conclusion
Although Group B showed a higher conversion rate than Group A, the p-value was greater than 0.05. Therefore, there is no statistically significant difference between the two groups based on the available sample.

Successfully completed an A/B Testing project using Python, Pandas, Matplotlib, and SciPy.
